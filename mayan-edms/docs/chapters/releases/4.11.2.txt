Version 4.11.2
==============

Released: April 13, 2026


Summary
-------

Version 4.11.2 is a maintenance release focused on restoring container-based deployments after the 4.11 series changes, tightening metadata type validation, and reducing unnecessary UI reload behavior.

Key highlights include:

- Docker Compose mirror services updated for 4.11 series compatibility.
- Docker entrypoint fixed to restore multi package APT and PIP install support.
- Top bar menu no longer reloads on every browser refresh.
- Metadata type parsing and validation components ensured to be present.
- Routine container image and Python dependency updates.


Changes
-------

Docker and deployment
^^^^^^^^^^^^^^^^^^^^^

Updated the Docker Compose `cabinet_mirror` and `index_mirror` services to work correctly with the 4.11 series changes. Fixed the Docker `entrypoint` to allow multi package APT and PIP installs to work again.


UI / UX
^^^^^^^

Fixed top bar menu so it no longer reloads on every refresh.


Metadata
^^^^^^^^

Ensured the metadata type parser and validator exist to prevent errors when defining or validating metadata types.
