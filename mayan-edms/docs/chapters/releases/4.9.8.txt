Version 4.9.8
=============

Released: December 27, 2025

Summary
-------

Version 4.9.8 is a maintenance release that merges changes from version 4.8.10 and backports selected improvements from version 4.10.3. It also updates several dependencies, refines upload wizard permission handling, and modernizes internal version checking and configuration tooling.

Changes
-------

Merges and Backports
^^^^^^^^^^^^^^^^^^^^

- Merge changes and improvements from version 4.8.10.
- Backport changes and improvements from version 4.10.3.


Dependencies
^^^^^^^^^^^^

- Update dependency versions:

  - django-formtools from version 2.3 to 2.5.1.
  - sentry-sdk from 2.46.0 to 2.48.0.
  - pypdf from 6.4.0 to 6.5.0.

- Modernize the `PythonDependency` class version checking logic and remove use of the deprecated `pkg_resources` library.


User interface
^^^^^^^^^^^^^^

Align cabinet and tag upload wizard step permission logic with the metadata wizard steps. The cabinet and tag steps are now skipped entirely when the user does not have access to any cabinets or tags.

Add a workaround for an undocumented backward incompatible change in `django-formtools` 2.4.0.

Tooling
^^^^^^^

Add the `config-env-copy` target as a forward compatible alias for the existing `copy-config-env` target.
