Version 4.9.9
=============

Released: April 22, 2026

Summary
-------

Version 4.9.9 is a maintenance release that merges changes from version 4.8.11 and backports selected improvements from series 4.11. It also updates several dependencies and merges security improvements.


Changes
-------

Dependencies
^^^^^^^^^^^^

Update Python dependency versions in addition to dependency version updates from Mayan EDMS 4.8.11:

- Markdown from 3.10 to 3.10.2
- Pillow from 12.1.1 to 12.2.0
- nh3 from 0.3.2 to 0.3.4


Error logging
^^^^^^^^^^^^^

- Added more descriptive module error log messages.
- Error log messages are now translatable.
- Error logs are now no longer cleared on success. Users must clear past
  error logs. This prevents a successful operation from clearing out still valid error logs from a previous operation.
- Increased the default error log retention per object from 3 to 15 entries.
- Added a subtitle to the object error log view showing the number of entries being retained for the specific object.


Settings
^^^^^^^^

- Split the smart settings app views into separate modules.
- Renamed the smart settings app view class names for uniformity.


UI/UX
^^^^^

Various help texts improvements.


Templating
^^^^^^^^^^

- Fixed the math template tag name `math_substract` to `math_subtract`. Update your existing templates if you use this template tag. An alias named `math_substract` was added for backwards compatibility but will be removed in a following minor version.
- Improved the math tags help texts.
