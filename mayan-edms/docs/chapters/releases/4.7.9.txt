Version 4.7.9
=============

Released: March 10, 2026


Summary
-------

This patch release consolidates maintenance updates from the 4.6.12 and 4.6.13 patch lines, including select backports from the 4.10 and 4.11 series. It also refreshes the `safety` dependency, tightens a search-related setting definition, standardizes internal Elasticsearch-related naming, and resolves documentation build warnings.


Changes
-------

Dependency & library management
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

- Updated `safety` from version 3.2.3 to 3.7.0.
- Refreshed dependencies versions.


Maintenance
^^^^^^^^^^^

- Merged improvements from versions 4.6.12 and 4.6.13, which include backports from the 4.10 and 4.11 series.
- Updated the project copyright year.


Search & settings
^^^^^^^^^^^^^^^^^

- Updated the setting `SEARCH_STORE_RESULTS_DEFAULT_VALUE` to be a choice field, improving validation and consistency of configuration.


Docker & developer tooling
^^^^^^^^^^^^^^^^^^^^^^^^^^

- Renamed internal references of `DOCKER_ELASTIC_` to `DOCKER_ELASTICSEARCH_` to align with the newer Elasticsearch naming convention.


Documentation
^^^^^^^^^^^^^

- Fixed documentation build warnings to improve Sphinx build cleanliness and stability.
