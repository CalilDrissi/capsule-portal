Version 4.7.10
==============

Released: April 19, 2026


Summary
-------

This maintenance release consolidates fixes and refinements from the 4.6.x line, including backported improvements originally introduced across the 4.9, 4.10, and 4.11 series. It also refreshes several Python dependencies and updates Docker image tags to keep the supported deployment stack current.


Changes
-------

Developer & Build System
^^^^^^^^^^^^^^^^^^^^^^^^

- Added a default `Makefile` target that performs no action and directs developers to use the `help` target for available commands.


Dependency & Library Management
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^

Updated Python dependency versions:

- CairoSVG: 2.8.2 to 2.9.0
- dateparser: 1.3.0 to 1.4.0
- elasticsearch: 7.17.12 to 7.17.13
- gevent: 25.9.1 to 26.4.0
- greenlet: 3.3.2 to 3.4.0
- gunicorn: 25.1.0 to 25.3.0
- pypdf: 6.7.5 to 6.10.2
- redis: 7.2.1 to 7.4.0
- requests: 2.32.5 to 2.33.1
- sentry-sdk: 2.54.0 to 2.58.0


Docker & Deployment
^^^^^^^^^^^^^^^^^^^

Updated Docker image tags:

- PostgreSQL: 14.18-alpine to 14.22
- RabbitMQ: 3.13.7-management-alpine to 3.13.7-management
